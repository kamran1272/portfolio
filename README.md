# Kamran Khan Portfolio

Personal portfolio website for Kamran Khan, a full-stack web developer focused on React, Laravel, and practical SEO improvements.

## Overview

This portfolio highlights:

- React and Laravel project work
- Service offerings for websites, dashboards, and SEO improvements
- Professional contact options and CV download
- Technical SEO basics such as metadata, structured data, sitemap, and robots rules

## Tech Stack

- React
- Vite
- Tailwind CSS
- DaisyUI
- React Router
- Font Awesome

## Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
```
